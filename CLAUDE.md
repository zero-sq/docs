# Space Zero Documentation

Documentation archive for Space Zero. Contains meeting notes, zerocap docs, and archived documents.

## Structure

- `archive/` — Historical documents
- `meetings/` — Meeting notes
- `zerocap/` — Zerocap-specific documentation

## Rules

- This is a documentation-only repo — no code, no tests, no builds
- Write in markdown
- Keep files organized by topic in the appropriate directory
