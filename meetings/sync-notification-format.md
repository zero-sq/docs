# Sync Meeting Notification Format

Syncs happen every 3 days. After each sync, meeting notes are saved here and a notification is posted to #team-general on Slack.

## Workflow

1. Save the meeting notes file to `meetings/` with naming convention `YYYY-MM-DD-sync.md`
2. Commit and push to main
3. Draft a Slack message for review before sending

## Slack Message Format

- Friendly opener noting the notes are saved to the docs repo
- Link to the file on GitHub
- Quick recap with bold bullet points summarizing key topics from the notes' Summary section
- Next steps as a checklist with proper Slack @mentions using user IDs (not plain names)
